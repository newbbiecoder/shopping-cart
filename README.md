A responsive shopping cart application built with **React** and **Tailwind CSS** as part of **The Odin Project** curriculum.

---
## Features

### Multi-Page Navigation  
- Home Page  
- Shop Page  
- Cart Page  
- Navigation bar displayed on all pages  
- Routing handled using **React Router**

### Shopping Cart  
- Add items to cart  
- Increase / decrease quantity  
- Remove items  
- Total price calculation

### Responsive Layout  
- Mobile-first design
